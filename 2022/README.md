# Advent of code 2022

## Requirements

stack: https://docs.haskellstack.org/en/stable/

## Run

All in one:

```shell
run s01
```

Manually:

```shell
stack build [s01]
stack exec s01
```

## Template

Use `generate.py` to generate a directory for a day given as command line argument.
This uses files in the `templates` folder.
